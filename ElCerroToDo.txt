REGISTRAR SERVIDORES COM
========================
Ejemplos:
Ema.exe /RegServer
Para windows 7 64 bits hay dos programas RegSvr32,
para 32bits está en C:\Windows\System32\regsvr32.exe
para 64bits está en C:\Windows\SysWOW64\regsvr32.exe
RegSvr32 Ema.exe
Para DESINSTALAR
Ema.exe /UnRegServer
RegSvr32 /u Ema.exe
Para servers ejecutables mejor usar:
COM.exe /RegServer
COM.exe /UnRegServer
Para componentes dll, ocx, etc usar:
RegSvr32 Ema.ocx
RegSvr32 /u Ema.ocx
